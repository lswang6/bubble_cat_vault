我帮你写一个 **一步到位的命令**，可以在你的 bubble_cat Vault 根目录执行，完成以下目标：

1. 删除仓库里已经被追踪的 .DS_Store、.obsidian/workspace.json 和 .obsidian/cache
    
2. 添加 .gitignore 文件（模板我上面给的）
    
3. 提交一次 commit
    
4. 保持仓库整洁


cd "/Users/louiswang/Library/Mobile Documents/iCloud~md~obsidian/Documents/bubble_cat"

# 1. 创建 .gitignore 并写入模板
cat > .gitignore << 'EOF'
# ==========================
# macOS 系统文件
# ==========================
.DS_Store
.AppleDouble
.LSOverride
._*

# ==========================
# Obsidian
# ==========================
.obsidian/workspace.json
.obsidian/cache/
.obsidian/plugins/obsidian-git/*.log
.obsidian/plugins/obsidian-git/tmp/

# ==========================
# 临时 / 编辑器文件
# ==========================
*.tmp
*.swp
*.swx
*.bak
*.orig
*.old

# ==========================
# iCloud 同步可能的临时文件
# ==========================
*.icloud
EOF

# 2. 删除已被 Git 追踪的垃圾文件
git rm --cached -r .DS_Store
git rm --cached -r .obsidian/workspace.json
git rm --cached -r .obsidian/cache

# 3. 添加 .gitignore 并提交一次
git add .gitignore
git commit -m "chore: add gitignore and remove tracked temp files"

# 4. 推送到远端
git push
